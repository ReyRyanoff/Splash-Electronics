# Splash Electronics

An e-commerce web application for browsing and purchasing computer hardware components like processors, graphics cards, and memory modules.

## Features
- **Product Catalog**: Browse electronics organized by category (Processors, Graphics Cards, Memory)
- **Shopping Cart**: Add/remove items, view cart summary with total price and product count
- **Responsive Design**: Mobile-friendly interface using Tailwind CSS and DaisyUI
- **Local Storage**: Cart data persists across browser sessions
- **Image Carousel**: Promotional banner on the home page with automatic slideshow

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Tailwind CSS, DaisyUI, Flowbite
- **Icons**: Font Awesome
- **Data**: JSON-based product API

## Project Structure
```
├── index.html          # Home page with product listing
├── cart.html           # Shopping cart page
├── api.json            # Product database
├── styles.css          # Custom styles
├── tailwind.config.js  # Tailwind configuration
└── images/             # Product images and assets
```

## How to Run
1. Open `index.html` in a web browser
2. Browse products and add items to cart
3. Click the cart icon to view and manage your shopping cart
4. Place order to complete checkout
